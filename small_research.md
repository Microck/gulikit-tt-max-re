TT MAX is a real official GuliKit controller, and GuliKit groups it with TT PRO on shared “TT MAX / PRO” product and firmware pages. Here are the main official TT MAX-related resources I found. ([GuliKit][1])

* **TT MAX / PRO Controller** — the official product page for the TT line, with specs, features, and supported platforms. This is the main official hardware page for TT MAX itself. ([GuliKit][1])

* **GuliKit Download** — the official downloads hub. In the search snippet I saw, it still surfaced **TT MAX / TT PRO firmware V6.7 dated 2026-04-03**, so it looks like the hub snippet can lag behind the newest dedicated firmware page. ([GuliKit][2])

* **TT MAX / TT PRO Controller V6.8 Firmware — 2026-04-13** — this is the newer dedicated TT firmware page. GuliKit says V6.8 fixes an occasional joystick-jitter issue when APG and rear-button mapping are mixed. The official download/news category also surfaces V6.8, so this is the TT MAX firmware page I’d trust first. GuliKit’s update instructions on this page say to power the controller off, connect with a USB-C to USB-A **data** cable, hold **APG + Home**, wait for a **“GuliKit” USB drive** to appear, unzip the firmware package first, then restart once after updating before checking the version. ([GuliKit][3])

* **APG File Editor** — official web-based APG editor. GuliKit explicitly says it is compatible with **TT Series** controllers that have firmware with APG support. ([GuliKit Macro Editor][4])

* **Test & Calibration** — official browser-based testing/calibration page. GuliKit says controller testing supports **all GuliKit controllers**, so this is an official TT MAX-adjacent utility page. ([GuliKit][5])

* **Setup Tool** — official browser-based configuration tool that works “without installing any software.” However, the supported-model list shown on the page currently includes **KK2T, KK3 Pro, KK3 Max, and Elves 2 Pro**, not TT MAX, so I would not assume TT MAX support there yet. ([GuliKit Setup Tool][6])

* **Support / Controller FAQ** — the official support hub plus GuliKit’s general controller FAQ page. These are not TT MAX-only, but they are the official support pages most likely to matter for TT MAX issues. ([GuliKit][7])

* **Official mobile app announcement** — GuliKit says an official **iOS/Android app** is scheduled for **Q2 2026**, with initial support including the **TT Series**. ([GuliKit][8])

Bottom line: the most useful official TT MAX resources right now are the shared **TT MAX / PRO product page**, the dedicated **V6.8 firmware page**, the **APG File Editor**, and the **Test & Calibration** page. From the official pages above, GuliKit’s TT MAX tooling currently looks mostly **web-based** rather than a standalone desktop utility, and the **Setup Tool** does not presently list TT MAX in its supported models.
